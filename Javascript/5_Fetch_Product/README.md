A simple web application that fetches and displays a list of users from a public API. Each user's details are shown in a card layout, including their name, contact info, address, and company details. Built with vanilla JavaScript, HTML, and CSS

## 🚀 Live Demo
Check out the live application here: [Live Demo Link](https://saylani-assignment-nu.vercel.app/Javascript/5_Fetch_Product/index.html)


## 🌟 Project Overview
This project demonstrates how to:
- Fetch data asynchronously from an external API
- Dynamically create and render HTML elements based on fetched data
- Style content responsively using CSS Grid and Flexbox

## ✨ Key Features
- **API Integration**: Fetches user data from [JSONPlaceholder](https://jsonplaceholder.typicode.com/users)
- **Dynamic Rendering**: Creates user cards for each entry in the API response
- **Responsive Layout**: Uses CSS Grid for a clean, adaptive display
- **Minimalist UI**: Simple, readable, and easy to extend

## 🛠️ Technologies Used
- **HTML5**: Markup structure
- **CSS3**: Layout and card styling
- **Vanilla JavaScript (ES6+)**: Fetching data and DOM manipulation
