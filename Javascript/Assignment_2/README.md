A modern and responsive digital clock created with HTML, CSS, and JavaScript. This project showcases fundamental web development skills, focusing on a clean UI, dynamic updates, and responsive design.

## Project Overview

This digital clock application displays the current time and date, updating every second. It was built as a portfolio project to demonstrate proficiency in front-end technologies and to create a visually appealing and functional UI component. The clock features a sleek, modern design with a digital-style font and a responsive layout that adapts to different screen sizes.

## Key Features

- **Real-Time Clock:** Displays the current hour, minutes, and seconds, updating every second.
- **Date Display:** Shows the current day, month, and year.
- **AM/PM Indicator:** Automatically determines and displays whether it's AM or PM.
- **Responsive Design:** The layout adjusts to various screen sizes, from mobile phones to desktops.
- **Modern UI:** A clean and visually appealing interface with a digital clock aesthetic.

## Project Structure

The project is organized into the following files:

-   `index.html`: The main HTML file that provides the structure for the clock.
-   `style.css`: The stylesheet that contains all the styling rules for the clock, including layout, fonts, and colors.
-   `script.js`: The JavaScript file that contains the logic for the clock, including getting the current time and updating the display.

## Learning Outcomes / Skills Demonstrated

-   **DOM Manipulation:** Using JavaScript to interact with and update HTML elements in real-time.
-   **JavaScript Timers:** Implementing `setInterval` to create a live-updating clock.
-   **CSS Styling:** Applying modern CSS techniques for layout, typography, and visual effects.
-   **Responsive Design:** Using media queries to create a responsive user interface that works on different devices.
-   **Project Structure:** Organizing files in a clear and maintainable way.

## Future Enhancements / Roadmap

-   **Theme Switcher:** Add a feature to switch between light and dark themes.
-   **Timezone Selector:** Allow users to select a timezone to display the time from different parts of the world.
