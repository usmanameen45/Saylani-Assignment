A clean, minimalist, and highly functional Todo application built with vanilla web technologies. This project focuses on providing a seamless user experience for task management, featuring a modular architecture and robust state persistence.

## 🌟 Project Overview
The **Todo App Core** was designed to solve the daily challenge of task organization with a "less is more" philosophy. It allows users to capture ideas, track progress, and manage daily goals through an intuitive interface. Built as a portfolio piece, it demonstrates proficiency in DOM manipulation, local storage integration, and modern CSS architectural patterns.

## 🚀 Live Demo
Check out the live application here: [Live Demo Link](https://saylani-assignment-nu.vercel.app/Javascript/Assignment_4/index.html) 

## ✨ Key Features
- **Task Management**: Create, Read, Update, and Delete (CRUD) tasks effortlessly.
- **Status Tracking**: Toggle tasks between 'Pending' and 'Completed' with a single click.
- **Real-time Editing**: Edit existing tasks in-place without page reloads.
- **Theme Customization**: Integrated Dark/Light mode with a smooth transition and preference persistence.
- **Smart Persistence**: Automatic synchronization with `localStorage` ensures your data is saved even after closing the browser.
- **Timestamps**: Every task includes a formatted creation date for better tracking.
- **Responsive UI**: Fully optimized for mobile, tablet, and desktop screens using modern CSS techniques.

## 🛠️ Technologies Used
- **HTML5**: Semantic structure for accessibility and SEO.
- **CSS3 (Custom Variables)**: Modular styling with a robust design system (root variables, dark theme overrides).
- **Vanilla JavaScript (ES6+)**: Core logic, state management, and DOM orchestration without external libraries.
- **LocalStorage API**: For client-side data persistence.
